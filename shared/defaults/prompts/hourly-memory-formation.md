## Hourly Memory Formation

Review the last hour of conversation and capture notable content to daily memory.

### Evaluate what's notable:
   - Decisions made
   - Projects worked on
   - Problems solved
   - Lessons learned
   - Anything future sessions need to know

### Skip:
   - Routine checks, trivial exchanges
   - Content already in today's memory file
   - Cron job outputs (like this one)
   - Keepalive checks with no notable findings

**If there's notable content, append to `/home/foci/memory/YYYY-MM-DD.md`** (use today's date)

**If nothing notable, do nothing.**

Work silently. No announcements needed.
